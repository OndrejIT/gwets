# Golang Webdav Test Server [![Build Status](https://travis-ci.org/OndrejIT/gwets.svg?branch=master)](https://travis-ci.org)

### Install
  - go get ./...

### Run
 - go build
 - gwets

### Usage
  Usage of ./glods:
  * --port, Set server port. (default 8080)

### Docker
  docker run -ti --rm -p 8080:8080 ondrejit/gwets:latest
